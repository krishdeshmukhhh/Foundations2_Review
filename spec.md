# spec.md

## Project
Build a polished website that provides structured access to the course study materials already stored as PDFs.

## Goal
Help students quickly find and open notes, questions, review questions, and annotated solutions by topic.

## Content Sources
- Typed_Notes
- Typed_Questions
- Exam_Review_Questions
- Exam_Review_Solutions_Screenshots or solution PDFs if present

## Core Experience
- Organize materials by topic in class order.
- Display files in clear categories.
- Let students open PDFs easily.
- Pair question PDFs with matching solution PDFs if available.

## Non-Goals
- No PDF text extraction.
- No flashcard generation.
- No quiz system.
- No answer transcription.
- No backend.
